Question-1: What are the different ways to select an element in the DOM?

Answer: getElementsByTagName(), getElementsByClassName(), getElementById(), querySelector(), querySelectorAll() - These five methods use to select an element in DOM.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML allows to add directly html code in JavaScript. On the other hand, innerText & textContent use to add text to an element. Both method works same but there is a key difference. Despite of display value hidden textContent shows the value but innerText doesn't.

Question-3: What is event delegation in the DOM?

Answer: When parent element handles event for its child element is called Event Delegation.

Question-4: What is event bubbling in the DOM?

Answer: Event Bubbling directs an event to its target. when event handler is trigged then it bubbles up from the childNode to parentNode.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: To create, add & remove element in JavaScript I will use createElement() method, append() method, remove() in the childNode or removeChild() in the parentNode.
